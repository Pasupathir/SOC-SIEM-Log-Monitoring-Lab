# SOC SIEM Log Monitoring Lab

## Overview
This project demonstrates a Security Operations Center (SOC) log monitoring setup using Splunk.

Logs generated from a Kali Linux machine are forwarded to a Splunk server for monitoring and analysis.

## Tools Used
- Splunk Enterprise
- Splunk Universal Forwarder
- Kali Linux

## Architecture
Kali Linux (Log Producer)
        ↓
Splunk Universal Forwarder
        ↓
Splunk Server (Monitoring & Analysis)

## Outcome
Successfully created a SIEM monitoring environment and analyzed logs using Splunk queries.
